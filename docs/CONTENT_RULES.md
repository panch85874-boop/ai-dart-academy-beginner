# 內容規則

## 基本原則

- 使用台灣用語繁體中文。
- 題目要適合 10 到 60 歲使用者。
- 不要做成教科書。
- 不要堆技術名詞。
- 鼓勵語要清楚，不要幼稚。

## 不可捏造

題目不可捏造：

- 工具功能
- 平台政策
- 價格
- 模型差異
- 商店規則
- 官方課程身份

不確定的內容一律標示「需要驗證」。

## 隱私與安全

- 不收集個資。
- 不要求輸入真實姓名、電話、地址、帳號、密碼或 API key。
- 不串接外部 AI API。
- 不使用未授權圖片、音效或字型。
- 不放入未公開資料或敏感資料。

## 題目欄位

每題必須包含：

- `id`
- `level`
- `gameType`
- `question`
- `options`
- `answer`
- `hint`
- `explanation`
- `voiceText`
- `rewardText`
