# 貢獻指南

謝謝你願意協助 AI 飛鏢闖關學院｜初階版。

## 可以貢獻什麼

- 新增或修正題目
- 修正文案，讓 10 到 60 歲使用者更容易理解
- 改善手機或電腦介面
- 改善無障礙操作
- 改善語音文字
- 修正錯字、格式或文件

## 題目貢獻規則

題目資料位於 `data/beginner-levels.js`。

每題必須包含：

- `id`
- `level`
- `gameType`
- `question`
- `options`
- `answer`
- `hint`
- `explanation`
- `voiceText`
- `rewardText`

題目要符合初學者情境，避免堆技術名詞。

## 文案原則

- 使用台灣用語繁體中文
- 鼓勵語要清楚，不要幼稚
- 避免過度保證
- 不要宣稱任何工具一定有某功能
- 不確定的工具功能、平台政策、價格、商店規則，要標示「需要驗證」

## 介面改善原則

- 大字體
- 大按鈕
- 手機與電腦都能用
- 一個畫面不要放太多文字
- 不使用未授權圖片、音效或字型
- 不引入大型框架
- 不新增登入、後台、資料庫或外部 AI API

## Pull Request 建議

1. 說明你改了什麼。
2. 說明如何驗證。
3. 如果改題目，列出影響哪些關卡。
4. 如果改介面，請附上手機與電腦檢查結果。
